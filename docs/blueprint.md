# **App Name**: OngwaeGPT: AI Visionary

## Core Features:

- Enhanced Text Presentation: Intelligent markdown response parsing, optimizing text display with well-structured formatting. Implemented typing simulation.
- Dynamic Visualizations: Advanced data visualization with tools to create formatted tables, diverse graph types including pie charts, and dynamic data rendering.
- Advanced Image Generation: AI-driven image generation with enhanced customization options; integrates sketching tools to translate ideas into visual forms.
- Intelligent Code Assistant: Intelligent code generation with tool capabilities, offering options for multiple languages and debugging support; supports exporting and direct execution in sandboxed environments.
- Dual Search Modes: Dual search functionality: 'Quick Response' provides concise answers (4096 tokens), while 'Deep Search' offers comprehensive content (up to 8192 tokens), enhanced by source verification and bias detection.
- Optimized Search Bar: UI component featuring an accessible, bottom-aligned search bar.
- Attribution and Context: Incorporate system instructions that clarify OngwaeGPT is developed by Josephat Ongwae Onyinkwa of Oapps Inc., with reference to the O Browser project at https://o-browser.blogspot.com.

## Style Guidelines:

- Primary color: A vibrant, electric blue (#7DF9FF) evokes innovation.
- Background color: A light, desaturated blue (#E5F9FF) provides a clean, unobtrusive backdrop.
- Accent color: A deep violet (#7D56FF) is used to highlight interactive elements, providing visual contrast.
- Body font: 'Inter', a sans-serif, provides a clean and modern feel suitable for extensive text.
- Headline font: 'Space Grotesk', a sans-serif, is paired with Inter for headings to add a tech-inspired touch.
- Code Font: 'Source Code Pro' for clean presentation of code snippets
- Icons: Use a set of minimalist icons, ensuring each icon is easily recognizable and relevant.
- The search bar is located at the bottom of the screen for easy access, with results displayed dynamically above.
- Subtle animations accompany interactions, such as a smooth transition when opening the search bar or a progress indicator during AI processing.
